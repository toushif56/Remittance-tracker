📤 ডাউনলোড (Google Sheets CSV)
সব ট্রানজেকশন CSV ফরম্যাটে ডাউনলোড হবে। Google Sheets এ ওপেন করতে পারবেন এবং সেভ করতে পারবেন।
💡 ডাউনলোডের পর Google Sheets → File → Import দিয়ে ওপেন করুন
📁 আপলোড (ডেটা রিস্টোর)
আগের CSV ফাইল আপলোড করলে সব ডেটা ফিরে আসবে। বিদ্যমান ডেটা মুছে নতুন ডেটা লোড হবে।
📁 CSV ফাইল বেছে নিন
⚠️ আপলোড করলে বিদ্যমান ডেটা প্রতিস্থাপিত হবে
🔄 Google Sheets এ কীভাবে ব্যবহার করবেন:
  1. উপরে CSV ডাউনলোড করুন বাটনে ক্লিক করুন
  2. Google Sheets খুলুন → File → Import
  3. ডাউনলোড করা CSV ফাইল আপলোড করুন
  4. ডেটা দেখতে ও এডিট করতে পারবেন Sheets এ
  5. ডেটা রিস্টোর করতে এখানে CSV আপলোড করুন
